# 🤍 Per te 🤍

Un sito web romantico dedicato a chi ami. ❤️

## Caratteristiche ✨

- 💌 **Lettera animata** che si apre con un click
- 🤍 **Cuori che cadono** continuamente dalla parte superiore
- 🎵 **Musica di YouTube** integrata automaticamente
- 📸 **Foto come sfondo** personalizzato
- 💫 **Effetti romantici** e animazioni fluide
- 📱 **Responsive design** - funziona su tutti i dispositivi

## Come usarlo 🚀

1. Clona il repository
2. Apri `index.html` nel browser
3. Goditi la magia! 🌟

## Personalizzazione 🎨

- Cambia il testo dentro i `.box`
- Modifica il messaggio della lettera
- Sostituisci la foto nel tag `<img>`
- Cambia il link del video YouTube
